github page + hexo 搭建博客

建个破博客简直折腾

原来打算万网域名主机一站搞定的，看了下价格，有点小贵，还要备案，挺麻烦的 -_-

免备案的狗爹godaddy又一直买不了，不知道是不是代理的问题，一直卡在付款页面，于是搜了下，居然还能用github page+hexo搭个简单的博客，有很多优点，无限流量，支持markdown（而且还是免费的！！暴露本性了)，虽然是静态页面，但是想了一下，大概都满足现阶段需求了，于是开搞，配置都挺简单的，不过后来遇到了一个问题太蛋疼了

发现没反应，google后发现原来
